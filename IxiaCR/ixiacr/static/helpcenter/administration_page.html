Using Axon : Administration operations
Administration operations
On any page, click Administration to access the Administration operations.
Devices, Endpoints, and Tracks: Configure the default settings
On the Administration page, click Devices, Endpoints, or Tracks to change the default settings.
Change the password for the current account
On the Administration page, click Change Password, and type the old and new passwords.
Update Axon
The system lists your current revision and the newest available revision. Click BEGIN UPDATE to update Axon to the newest software revision.
System backup and restore
Click SYSTEM BACKUP AND RESTORE to back up or restore the current system settings, test definitions, and test reports to an AXONâ€‘format file.

To back up the current system settings
1
Click BACK UP THE SYSTEM
2
The backup process can take up to ten minutes. Do not close the browser or power down the Axon device during backup.
3
When backup finishes, click OK
You have the option to archive the backup file to an external drive as described next.
w
To archive backup files
In the Restore the system section, click ARCHIVE TO EXTERNAL DRIVE. When you confirm the actions, Axon selects the first available external drive and archives the file.
w
To restore the system settings from a backed-up file
You have the option to restore from the disk on the chassis (the local disk) or from an archived file on an external disk.
To restore from a backup file on the disk on the chassis:
In the Restore the system section, select the file and then click RESTORE
To restore from a backup file that is archived on a remote disk:
If a copy of the backup file no longer exists on the local disk, you must import the file before restoring:
a
In the Restore the system section, click IMPORT FROM EXTERNAL DRIVE and then select the file.
b
Click RESTORE
w
To delete backup files
You can delete backup files (.AXON format) from the disk on the chassis or from an archived file on an external drive.
In the Restore the system section, select the file and then click DELETE
