Using Axon : Test Results
Test Results
Use any of the following methods to view results:
n
On the Traffic Player page — during or after a run — click the Results tab
n
On the Dashboard, click the History tab and then select a report
n
On the Dashboard, select a report from the Recent Results section
Note While a test runs, Axon displays information that is returned by the network, but does not yet display detailed results.
While viewing a chart, use the slider to vary the scale of the X axis.
Interpreting test results
The description for each test type includes suggestions on what to look for in the results.
Total Bandwidth
Total Bandwidth is the total Layer 2 bandwidth seen by the test interfaces on the wire.
w
To “drill down” into particular values in test results
Hold the cursor over any data point to view the value at that moment in time.
w
To view a detailed summary of test results
Click Details to view a summary of all of the data collected while the test ran.
w
To print or save a detailed summary of test results
Click Details to view the data summary and then click one of the buttons to print the results or to save to a .csv format file.
