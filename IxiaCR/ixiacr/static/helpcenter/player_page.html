Quick start: Configure and run a test
The overall process of running a test on your network is:
1
Select the type of test that will give you the info you need
2
Configure the test: device and traffic settings and test duration
3
Save the test settings for use another day
4
Click the Run button to run the test
5
While the test runs (and afterward) look over the report for the info you need
Example
Step 1
Select the type of test to run
We click Run a Test to get started. The Run a Test page displays the types of tests that are built in to Axon. Each type includes several tests that are similar in function and tuned to return specific test results. You will choose the appropriate test from one of the groups. In our example, we select Network Infrastructure Test.
Tip If you are not sure, you can click Need help finding the right test. The wizard prompts you to describe your testing requirements and then presents a list of appropriate tests.
Axon lists all of the tests in the Network Infrastructure Test category and provides a quick description for any test that you click. Based on your needs, select one of the tests.
Step 2
Configure the test settings — as easy as 1, 2, 3!
On the Traffic Player page:
Click the ports to use
From the Endpoints tab, drag‑and‑drop the appropriate endpoint type into the appropriate chassis
Type the bandwidth for the test (some test types might offer another setting in this section)
Now you are ready to save and run the test.
Note There are many more settings that you can configure. See “Details for working on the Traffic Player page” on page 10.
Step 3
Save the test
If you expect to run the test again at another time or location (or even if you do not), click Save to save the configuration settings for the test.
Name
Specify a meaningful name for the test that represents the configuration settings that you made.
Favorite
Check the box to add the test for easy access on the Favorite Tests list on the Dashboard.
In addition, the test will appear when you select Favorites on the test configuration page:
Tags
To enable you to quickly access appropriate tests when you take Axon on the road, you might want to organize test configurations by customer, by location, or by a tag that you specify (for example, “large packets”). Specify a tag by typing into the Tag text box.
Step 4
Run the test
Click Run
The test executes and results appear on the Results tab.
Step 5
View the test report (as described in “Test Results” on page 13)
