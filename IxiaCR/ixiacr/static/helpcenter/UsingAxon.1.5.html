Using Axon : Test Types
Test Types
Axon organizes all tests into types. The types are displayed when you click Run a Test — Network Infrastructure tests, Applications tests, and Network Device tests. Each test type includes several tests that are similar in function and tuned to return specific test results.
•
To view a description of the intent, operation, and results of a test, select it while browsing tests. Alternatively, while working on a test configuration on the Traffic Player page, click the Documentation tab.
•
While trying to determine which test to run, you can click Need help finding the right test. The wizard prompts you to describe your testing requirements and then presents an appropriate list of tests.
