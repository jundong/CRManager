You can use Axon to test your network and proactively validate the performance of applications, network devices, and services throughout their lifecycles.
Axon provides a set of predefined tests. Each test implements a test methodology that is tuned to provide you with specific information about the various aspects of network performance. Regardless of the type of test, you follow a simple consistent configuration process, save the configuration for use another day, and then run the test.
Important The network connection on the Axon device enables access to the management interface for the device. To ensure only authorized access, the device must be on a secure network.
